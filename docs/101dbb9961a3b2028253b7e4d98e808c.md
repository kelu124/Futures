## Summary

Irchiver is a software project designed to capture and store web content automatically and continuously for users, primarily running on Windows. It aims to provide a backup memory of web pages viewed, allowing users to access their browsing history without relying on cloud storage, thus prioritizing privacy. Unlike other archiving tools, Irchiver captures both images and text in a local format, ensuring long-term accessibility. The software is currently in development with various planned updates to enhance efficiency and functionality. Users are encouraged to subscribe for updates and can request the source code under specific conditions.
