## Summary

This guide provides OSINT researchers with a comprehensive overview of Discord, a communication platform originally created for gamers. With 563 million registered users, Discord has evolved into a hub for various interests, making it a critical resource for investigations. The article discusses Discord's centralized architecture, server structure, and how to find both listed and unlisted servers. It highlights tools and techniques for searching within Discord, including Google dorks and the platform's built-in search functionalities. Additionally, the guide explains how to investigate user profiles and gather information from external resources. Ultimately, Discord serves as a valuable asset for OSINT research, offering insights into user interactions and community dynamics.
