## Summary

Le texte explore une nouvelle rythmicité dans les écrits numériques, où deux figures de style dominent : le "diptyque pivot" et le "triptyque rythmique". Le diptyque pivot juxtapose des idées contrastées (ex : « Ce n’est pas ceci, c’est plutôt cela »), créant des attentes avant de les renverser. Le triptyque rythmique utilise une séquence de trois propositions qui s'amplifient (ex : « C’est un système qui contraint, qui renforce, qui verrouille »). Ces structures stylistiques se rencontrent dans divers contenus, notamment sur les réseaux sociaux, illustrant comment les discours sur des figures influentes comme Sam Altman, Elon Musk et Peter Thiel intègrent ces éléments pour renforcer leur impact narratif.
